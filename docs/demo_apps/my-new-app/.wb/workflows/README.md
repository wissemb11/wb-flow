# Coding Rules: my-new-app

1. **Composition API Only:** Never use the Options API. All components must use the modern `<script setup>` syntax.
2. **No Direct DOM Access:** Rely purely on Vue's reactivity and template refs. Do not use `document.getElementById` or raw CSS selectors.
3. **Strict Scoped Styling:** All `<style>` blocks in `.vue` files must include the `scoped` attribute to prevent global style bleed.
4. **Vite Environment Standards:** Use Vite's `import.meta.env` for environment variables. Never use `process.env`.
