## Quick Start

1. **Run `/wbSetup .`** in your project root to initialize wb-flow
2. **Run `wbValid --quick`** to verify the setup
3. **Run `/wbContext`** to load project context for your LLM session
4. **Explore commands** by family: [Planners](../commands/README.md#planners), [Workers](../commands/README.md#workers), [Critics](../commands/README.md#critics)
