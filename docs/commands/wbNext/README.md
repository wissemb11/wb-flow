# wbNext — Canonical Examples

Below are the optimal invocation patterns for `/wbNext`.

### Standard Execution
```bash
/wbNext packages/target
```
### Suggests the next bug to fix.
```bash
wbNext --focus bugs
```

### Lists quick-win tasks under 30 minutes.
```bash
wbNext --quick-wins
```
